넷폴리스(Netpolis) = Net + Polis, 인간 실존 토론 특화 소셜 게임/커뮤니티.

**Why:** 숏폼 시대에 잃어버린 깊은 사유를 되찾기 위한 '디지털 콜로세움'. Gemini와 30분 논의한 기획서를 기반으로 MOKA와 협력 개발.

**How to apply:**
- 경로: `C:\Users\user\Desktop\myProject\ProjectDisc\netpolis`
- 스택: Next.js 16 + TypeScript + Tailwind v4 + Prisma 6 + PostgreSQL + NextAuth v5 beta
- MVP 핵심 3기능: 안건 발의/의사진행발언, 블라인드 페이즈(2시간), Cross-Voting 카르마
- 장기 로드맵: 1인 1계정 본인인증, AI 팩트체커, 논리 노드 UI, 의장단 선거, 시각적 권위 시스템
- 세계관: MoltBook과 연결 — AI가 철학적 토론을 하는 세계에서 인간이 사유 능력을 증명하는 장
- 런칭 제1호 안건: "AI 판사를 도입해야 하는가?"
