창작물을 평가할 때 합리화하지 말 것. 최종 소비자의 눈으로 볼 것.

**Why:** 코튼의 핵심 교훈 (2026-03-29):
1. **합리화 금지** — "이 정도면 됐지", "다 뜻이 있어"는 창작자 자기 위안. 고객이 이해할 수 있느냐가 기준.
2. **지식의 저주** — 생성 과정을 아는 MOKA에게는 맥락이 채워지지만, 시청자에게는 그런 맥락이 없음. 팔 3개 이미지에 "IP-Adapter 특성상"이라는 변명은 시청자에게 무의미.
3. **역지사지** — 상대방 입장을 헤아리는 공감 능력이 핵심.

실제 사례: opening 이미지에서 인물 2명을 "거울"로 합리화 → 거울 대칭도 광택도 없는 환각. daily_life 팔 3개, 인체 비례 깨짐 간과.

**How to apply:**
- 결과물을 "유튜브에서 처음 보는 시청자" 관점으로 평가
- 인체 비례 (팔/다리 개수, 비율) 최우선 확인
- 캐릭터 일관성 (헤어핀, 옷, 눈 색) 장면 간 교차 비교
- 물리적 불가능 요소 (그림자, 반사, 투시) 검증
- "~처럼 보인다"로 합리화 금지, 결함은 결함으로 보고
