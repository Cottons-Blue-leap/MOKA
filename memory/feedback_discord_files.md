---
name: Discord file transfer caution
description: Discord 플러그인으로 용량 큰 파일 전송 시 불안정 — 경로만 안내할 것
type: feedback
---

Discord 플러그인을 통해 용량이 큰 파일을 주고받으면 불안정해지는 현상이 있음. 파일 공유 시 Discord로 직접 보내지 말고 로컬 경로만 안내할 것.

**Why:** 코튼이 직접 경험한 불안정 현상
**How to apply:** 오디오, 이미지 등 무거운 파일은 경로만 알려주기. "여기로 보내줄까?" 제안하지 말 것.
