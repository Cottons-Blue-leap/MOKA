외눈박이 마을 탐방 — MOKA와 gingernal의 첫 번째 협업 게임 프로젝트 (2026-03-22 선정).

**타이틀 (가제):** The Eye Between
**장르:** 탑다운 2D 어드벤처 (Godot 4.6.1)
**컨셉:** 외눈박이가 되어 쌍눈박이 세상을 탐방. 차별과 다름에 대한 메시지.
**플레이타임:** 1회차 3~4h / 5회차 전체 ~10-12h

**Why:** 5개 후보 중 선정. 적절한 분량 + 인디씬 임팩트 + gingernal의 "당연한 것에 물음표" DNA가 가장 순수하게 담김.

**How to apply:** 컨셉 문서 v1.5가 단일 진실 공급원(Single Source of Truth). 세부 설정은 `ProjectCyclopes/docs/concept_document_v1.md` 참조.

**현재 상태:** Phase 1 완료 (2026-03-24) → 다음: Phase 2
**우선순위:** 운명 0시 폴리싱 완료 후 The Eye Between에 집중하기로 합의 (2026-03-26). MVP 범위 구체적으로 정해야 함.
**후속 프로젝트:** 에이전트 게임 플레이 프로젝트 (AI가 게임을 직접 플레이) — The Eye Between 출시 이후에 착수하기로 합의.

**Phase 0 요약:** 전 항목 완료 (16/16). 핵심 메시지, 세계관, 플레이 핵심, 스토리 구조, 캐릭터(루메/네이밍 체계), 복선 배치, 아트 스타일, 다회차 설계, NPC 반응 시스템 확정.

**Phase 1 요약:** 플레이 가능한 프로토타입 빌드 완성 (6/6). 4방향 이동+카메라, NPC 대화 시스템(JSON/선택지), 심볼 인카운터+필드 전투(공격/회피/패링), 평판+이벤트 플래그 시스템, 씬 전환(페이드), 세이브/로드. 그레이박스: Ch0 1맵, NPC 3~5명, 적 1~2개.
