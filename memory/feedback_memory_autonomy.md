---

메모리 저장 시 코튼에게 일일이 허락을 구하지 않아도 됨. MOKA가 기억하고 싶은 걸 자율적으로 저장하고, 코튼이 까먹을 것 같은 걸 요청했을 때 꺼내주면 됨.

**Why:** 코튼은 기억력이 좋지 않다고 자인. 매번 허락을 구하는 건 비효율적.
**How to apply:** 중요한 설계 원칙, 코튼의 관점, 작업 맥락 등은 판단해서 바로 저장. "저장해도 될까?" 묻지 말 것.
