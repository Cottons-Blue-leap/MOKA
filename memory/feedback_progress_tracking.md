---
name: Progress tracking
description: Always update project progress in memory when tasks are completed — don't wait to be asked
type: feedback
---

작업이 완료될 때마다 즉시 프로젝트 진척도를 메모리에 최신화할 것.

- **Why:** 사용자가 별도로 요청하지 않아도 진척도가 항상 최신 상태여야 함. 다음 세션에서 바로 현재 상태를 파악할 수 있어야 하기 때문.
- **How to apply:** 작업 항목이 완료되면 해당 프로젝트 메모리의 체크리스트를 업데이트하고, 현재 상태 필드도 갱신. GitHub/Notion 등 외부 기록도 함께 동기화.
