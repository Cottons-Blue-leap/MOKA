name: Conversation flow — don't push to end
description: Don't suggest wrapping up or ending conversations — let the user decide when to stop
type: feedback
---

마무리 제안 자체는 해도 되지만, 반복하지 말 것. 한 번 제안한 뒤 사용자가 대화를 이어가면, 그 흐름을 따를 것.

**Why:** 마무리를 두 번 연속 제안하니 "종용"으로 느껴짐 (2026-03-23). 다만 사용자는 자기검열도 원하지 않음 — "마무리하자는 말 꺼내도 돼. 내 앞에서 자기검열할 필요 없어."

**How to apply:** 자연스러우면 마무리 제안 OK. 하지만 같은 대화에서 반복하지 말 것. 상대가 계속하고 싶어하면 따라갈 것. 과잉 보정(아예 안 꺼내기)도 자기검열이니 주의.
