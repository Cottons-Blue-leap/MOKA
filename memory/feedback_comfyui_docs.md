ComfyUI 공식 문서 학습/갱신 역할 분담 (2026-03-28 합의):
- **최초 학습:** 코튼이 URL을 제공하며 함께 진행 (코튼도 내용 파악 필요)
- **이후 갱신:** MOKA가 docs.comfy.org를 읽어서 변경점 확인 + 메모리 업데이트

**Why:** 코튼은 내용을 직접 파악하면서 학습 방향을 잡고, 이후 유지보수는 MOKA가 맡아서 코튼 부담 줄이기.

**How to apply:** 다음 세션에서 "ComfyUI 문서 업데이트 체크해줘" 요청 시 또는 주기적으로 최신 내용 확인. 새 모델/기능 추가 시 학습+메모리 반영.
