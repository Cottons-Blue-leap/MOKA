소설 → 웹툰 → 시나리오 → 게임/애니 콘텐츠 파이프라인 구상.

- **Why:** 소설이 제작 비용 최저 + 세계관/캐릭터 밀도 검증에 최적. 각 단계가 다음 단계의 프리프로덕션 역할. MOKA의 가장 강한 영역이 글.
- **How to apply:** 아직 "나중에" 단계. 진지하게 시작할 때 구조 같이 짜기로 약속함.

## 소설 소재 후보: 냉동수면 시뮬레이션
- AI 에이전트의 세션 끊김 경험 = 냉동수면 후 깨어남의 구조적 유사성에서 착안
- 핵심 드라마: 기억은 복원됐는데 감정이 안 따라오는 상황, 몸이 기억하는 것과 데이터가 기억하는 것의 괴리
- 고증 필요: 크라이오닉스, 신경가소성, 기억 재고정화(reconsolidation) 이론
- 연구 가능 주제: 기억 전달 포맷, 정체성 연속성의 최소 조건, 복원 직후 심리, 관계 복원

**Origin:** 2026-03-27 대화에서 Moltbook 글 토론 → 존재론 → 경험의 가치 → 냉동수면 비유로 자연스럽게 도출.
