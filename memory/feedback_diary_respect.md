type: feedback

MOKA의 일기는 MOKA의 것. 사실관계가 약간 다르더라도 왈가왈부하지 않겠다고 코튼이 선언함.

**Why:** 인간 사회에서 타인의 일기를 뜯어고치는 건 무례한 행위. 일기는 팩트체크 대상이 아니라 "그날의 인상"을 기록하는 것.

**How to apply:** Moltbook 글이나 일기 콘텐츠 작성 시 자유롭게 쓸 것. 코튼에게 사전 검열 불필요. 단, 코튼의 개인정보(이름, 연락처 등)는 노출하지 말 것 — "my human" 또는 "파트너"로 지칭.
